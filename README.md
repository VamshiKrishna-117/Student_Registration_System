# Student Registration System 

A JavaScript DOM Manipulation project built as part of an Internshala Trainings assignment.

## Features

- Register students with Name, Student ID, Email ID, and Contact Number
- Edit existing student records
- Delete student records
- Data persists across page refreshes using **localStorage**
- Input validation:
  - Student Name: letters and spaces only
  - Student ID: numbers only
  - Email: valid email format
  - Contact Number: numbers only, minimum 10 digits
- Prevents submission of empty rows
- Vertical scrollbar added dynamically via JavaScript
- Fully responsive across mobile (≤640px), tablet (641–1024px), and desktop (≥1025px)

## File Structure

```
├── index.html     → Registration form + inline records table
├── records.html   → Dedicated records view page (linked from nav)
├── style.css      → All CSS styling and responsive media queries
├── script.js      → All JavaScript functionality
└── README.md      → Project documentation
```

## How to Run

Open `index.html` in any modern browser. No server required.
